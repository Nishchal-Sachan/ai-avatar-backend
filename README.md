# AI Avatar Platform Backend

Production-ready Node.js backend with Express, MongoDB, and JWT authentication.

## AI Providers

- **LLM** → Groq (llama3-8b-8192)
- **Embeddings** → HuggingFace Inference API (sentence-transformers/all-MiniLM-L6-v2)

OpenAI is now optional and no longer used for LLM or embeddings.

## Getting Started

See `.env.example` for required environment variables.
